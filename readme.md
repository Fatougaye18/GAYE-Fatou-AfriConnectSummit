# AfriConnect Summit 2026 — Site vitrine

**Nom :** Fatou Gaye
**Classe :** L1 DSBD

## Description

Site vitrine pour "AfriConnect Summit", un événement tech panafricain fictif réunissant
développeurs, entrepreneurs et investisseurs, à Dakar (Sénégal). Le site comporte 4 pages :
accueil, programme, intervenants, inscription/contact.

## Technologies utilisées

- HTML5 sémantique
- CSS3 (variables CSS, Flexbox, Grid, animations, media queries)
- JavaScript vanilla (aucun framework)
- Google Fonts (Space Grotesk, Inter)
- Bootstrap Icons

## Fonctionnalités JavaScript implémentées

1. Dark mode / Light mode avec persistance localStorage
2. Navbar dynamique au scroll + menu hamburger mobile
3. Animations au scroll (IntersectionObserver)
4. Compte à rebours en temps réel
5. Compteurs animés sur les chiffres clés
6. Onglets du programme (3 jours)
7. Filtrage dynamique des intervenants par thématique
8. Validation complète du formulaire d'inscription
9. Bouton retour en haut
10. Année dynamique dans le footer

La FAQ de la page Contact est un accordéon 100% CSS (aucun JavaScript).
